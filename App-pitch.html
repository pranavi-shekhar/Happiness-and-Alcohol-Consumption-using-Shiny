About the App

This app uses the Happiness and Alcohol Consumption data set obtained from Kaggle to build a linear regression model for observing the average happiness score of individuals across 122 countries, as a function of consumption of different alcoholic beverages as well as the Human Development Index(HDI).

The plots created are not particularly useful for making predictions, but can be used for observing and understanding the trends in and the structure of the data set.

Link to Application on Shiny Server

Description of the Data

  • This data set aims to explore the happiness levels of individuals based off of their per capita consumption of alcoholic beverages , Human Development Index(HDI) and GDP.

  • Happiness Score is measured on a scale of 1-10 with 10 being the happiest. The data set shows the average happiness score indicated by the countrys residents.

  • The HDI used is provided by the United Nations. Further, the region and hemisphere of each country has been specified. The unit for per capita consumption of beverages is Liters. Again, the country wise average is shown.

  • More details regarding the data set can be found here
